How It Works

Multi-Provider Failover

When you search, the addon tries providers in priority order. If Groq is rate-limited, it instantly falls back to Cerebras, then Google, then Cloudflare. You never see an error.

Query Intelligence

Exact titles like "The Matrix" go directly to Cinemeta (zero AI cost). Natural language like "movies about AI" uses AI semantic search. This saves ~40% of API calls.

Series Support

Unlike v1, this version properly handles TV series. The AI searches for series, and metadata is fetched from the series endpoint — not the movie endpoint.
